# 项目开发历史

## 项目概述
**项目名称**: 配置解析可视化工具  
**项目描述**: 智能解析配置JSON，可视化显示各个源的位置和覆盖范围，让配置一目了然

## 技术栈
- **前端**: HTML5 + CSS3 + JavaScript (原生)
- **样式**: 渐变背景、现代化卡片设计
- **图标**: SVG + PNG fallback
- **数据**: CSV格式进程库
- **版本控制**: Git

## 项目结构
